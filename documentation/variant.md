## Create a variant

### Steps to create a variant

```bash
POST /api/v1/variants
```

#### Request body

| Name                  | Type                | Description                               |
| --------------------- | ------------------- | ----------------------------------------- |
| `name`                | `string`            | -                                         |
| `description`         | `string`            | -                                         |
| `product`             | `string`            | -                                         |
| `images`              | `array` of `string` | -                                         |
| `size`                | `string`            | -                                         |
| `color`               | `string`            | -                                         |
| `slug`                | `string`            | -                                         |
| `sku`                 | `string`            | -                                         |
