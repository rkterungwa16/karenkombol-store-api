## Create a category

### Steps to create a category

```bash
POST /api/v1/categories
```

#### Request body

| Name                  | Type                | Description                               |
| --------------------- | ------------------- | ----------------------------------------- |
| `name`                | `string`            | - Clothing types: `dress`, `shirts`, `skirts`, `trousers`.                                        |
